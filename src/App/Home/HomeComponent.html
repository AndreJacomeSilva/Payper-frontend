FAQ

O Payper é um sistema informático que analisa as faturas enviadas pelos clientes, extrai apenas a informação necessária para fornecer os serviços de comparação e gestão, e armazena uma cópia da fatura num servidor encriptado. Apenas as faturas que não poderem ser automáticamente processadas, poderão vir a ser analisadas por um operador, para melhorar os processos de extração de informação. Os tarifários de eletricidade e gás disponibilizados no Payper são obtidos através da entidade reguladora do mercado, a ERSE. Os tarifários de telecomunicação são obtidos através dos portais dos diferentes operadores. Os tarifários da água são obtidos através do processamento das faturas da comunidade. O Payper é uma ferramenta automática que agiliza o processo de migração de fornecedor. Apenas após ter confirmado a sua intenção de mudar para um fornecedor recomendado, o Payper envia, de forma eletrónica, todos os dados necessários para o respetivo fornecedor, o qual poderá vir a entrar em contacto consigo caso seja necessária mais alguma informação ou documento. O Payper mostra sempre qual é o tarifário mais barato do mercado. O tarifário recomendado é baseado não só no preço, como também pela votação do resto da comunidade e pontuação pela própria equipa do Payper.
Cada vez que um novo tarifário é lançado no mercado, o Payper analisa todos os clientes que tenham uma fatura introduzida com menos de 6 meses e, caso a poupança anual seja superior a 10 Euros, envia um alerta através de email e Push Notification a notificar a existência de um tarifário mais barato. Pode inserir quantas faturas desejar. Aliás, para a comparação de tarifário de eletricidade e gás, o ideal é ter pelo menos duas faturas, com um ano de intervalo - para podermos analisar a sua distribuição de consumo ao longo de um ano completo. As faturas com mais de 3 anos serão automaticamente eliminadas para poupar espaço. O Payper permite que insira as suas faturas diretamente, usando a App, fazendo o forward do email da fatura para um endereço específico ou, usando um dos plugins disponibilizados para o Gmail ou Hotmail de reencaminhamento automático.

O que é o Payper ?


O Payper é o resultado de um projeto de inovação lançado por uma comunidade de profissionais nas áreas de energia, telecomunicação e software.

Nasceu do know-how no processamento de grande quantidade de dados (BigData) e análise de dados de faturas (Artificial Inteligence), este projeto visa disponibilizar, de forma gratuita, um gestor de faturas acessível para toda a gente.
